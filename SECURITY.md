# Security Policy

## API Key Handling

- Never commit your `STEMSPLIT_API_KEY` to version control
- Use environment variables or a secrets manager
- Rotate keys at [stemsplit.io/dashboard](https://stemsplit.io/dashboard) if compromised

## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Email **security@stemsplit.io** with details. We respond within 48 hours.

## Supported Versions

We support the latest stable release. Upgrade promptly when new versions are released.
